# Website Redesign Brief: Audrey Morgan Lee

## Project Overview

**Client:** Audrey Morgan Lee
**Current Domain:** https://audreymorganlee.com/
**Project Type:** Personal Author Website Redesign
**Location:** Philadelphia, Pennsylvania

---

## Objectives

1. Create a modern, visually compelling author website that reflects Audrey's literary voice
2. Improve navigation and user experience across all content areas
3. Establish a clear information architecture with dedicated pages for each content type
4. Showcase the forthcoming short story collection *Utter Goodness* prominently
5. Make publications, awards, and media easily discoverable
6. Maintain professional credibility while expressing creative personality

## Design Considerations

### Visual Direction
- **Tone:** Literary, sophisticated, approachable, contemporary
- **Typography:** Clean serif for headings (literary feel), readable sans-serif for body text
- **Color Palette:** Consider muted, sophisticated tones; avoid overly bright or busy colors
- **Imagery:** High-quality author photos; book covers; minimal decorative elements

### Technical Requirements
- Mobile-responsive design
- Fast loading times
- SEO optimization for author name and book titles
- Social media meta tags (Open Graph, Twitter Cards)
- Accessible design (WCAG compliance)
- Easy content management (WordPress or similar CMS)

### Integrations
- Substack newsletter embed/link
- Social media feeds or links
- Book retailer links (Bookshop.org, Amazon, publisher direct)
- Media embeds (Spotify, Apple Podcasts, YouTube)
